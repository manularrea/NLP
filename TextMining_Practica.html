def translate_word_by_word(text, source_lang='es', target_lang='en'):
    # Tokenizar el texto en palabras
    words = word_tokenize(text, language='spanish')
    
    # Traducir cada palabra individualmente
    translated_words = [GoogleTranslator(source=source_lang, target=target_lang).translate(word) for word in words]
    
    # Volver a unir las palabras traducidas en una sola cadena
    translated_text = ' '.join(translated_words)
    return translated_text

class WordByWordTranslator:
    def __init__(self, source_lang='es', target_lang='en'):
        self.source_lang = source_lang
        self.target_lang = target_lang

    def set_target_language(self, target_lang):
        self.target_lang = target_lang

    def translate(self, text):
        words = word_tokenize(text, language='spanish')
        translated_words = [GoogleTranslator(source=self.source_lang, target=self.target_lang).translate(word) for word in words]
        translated_text = ' '.join(translated_words)
        return translated_text

Qu hacemos con el error que se obtiene al meter una palabra que no est en el vocabulario? Solucionar este error es parte de la prctica de la asignatura.

Cuando una palabra no est en el vocabulario generado durante la fase de entrenamiento, no puede ser integrada en el vector de una frase que la contenga. Incluirla requerira recalcular el diccionario completo, provocando su crecimiento constante, ya que siempre aparecern trminos nuevos segn el texto en el que se haya basado dicho vocabulario.

En estos casos, se decide formar el vector de la frase ignorando la nueva palabra. No es recomendable generar un error o detener el proceso, ya que una nica palabra en una frase puede no cambiar completamente el significado y, aun sin representarla, se podra obtener una prediccin adecuada. Sin embargo, esta eleccin depende del criterio del implementador y no puede considerarse absolutamente correcta o incorrecta.

Para mantener la transparencia, es necesario emitir una advertencia cuando se omite una palabra desconocida. Esta prctica permite que el anlisis contine sin interrupciones y que el usuario del modelo est al tanto de posibles limitaciones en el anlisis.
